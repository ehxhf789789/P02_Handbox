첫 번째 지시 (환경 셋업)

이 프로젝트는 건설신기술 평가 시스템 데스크톱 앱이다.
Tauri (Rust + React/TypeScript) + Python FastAPI 구조로 되어 있다.

개발 환경을 셋업해줘:
1. cnt-agent-studio/ 에서 npm install
2. cnt-agent-studio/src-tauri/ 에서 Rust 빌드 확인
3. .env.example을 .env로 복사하고 AWS 키 설정
4. npm run tauri dev 로 앱 실행 확인
두 번째 지시 (프로젝트 파악)

프로젝트 구조를 파악해줘.
핵심 파일:
- cnt-agent-studio/src/stores/workflowStore.ts - 워크플로우 실행 로직
- cnt-agent-studio/src/components/ - UI 컴포넌트
- cnt-agent-studio/src-tauri/src/ - Rust 백엔드 (AWS SDK)
- aws_agent/api/ - Python FastAPI 서버
이후 개발 지시 (원하는 기능)
